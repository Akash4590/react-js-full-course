# Installation
To run this project locally:
### 1 Clone the repository
```bash
git clone https://github.com/Akash4590/react-js-full-course.git
```
### 2 Navigate to the project folder
```bash
cd react-course
```
### 3 Install dependencies

```bash
npm install
```
### 4 Run the development server
```bash
npm run dev
```
